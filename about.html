Cardboard Butler

What is this?

Cardboard Butler is an open-source web page, that helps you decide the next boardgame to play.

Why?

As boardgame collections grow, it gets more and more difficult decicde on what to play. This will help you decide.

Who?

Cardboard Butler is created and maintained by Philip Kristoffersen

How can i help

If you like this page, send me a message, and let me know that you like my work.

All game data is from BoardGameGeek. So you should consider supporting BGG here, as Cardboard Butler would not be possible without BGG.

If you are a developer, head over to Cardboard Butler Github Page and code away.
